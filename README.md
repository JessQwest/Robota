# Robota

Robota is a hyperspecific Discord bot dedicated to the needs of my forgetfulness, as well as everyday silliness. The main purposes are listed below:

## Better Embeds

Specify a channel in the config, and video links from Twitter/X, TikTok and Bluesky will be replaced with better embeds.

## Calendar System

Specify a webdav server hosting your outlook calendar, and Robota will remind you of your upcoming events with daily summeries.
